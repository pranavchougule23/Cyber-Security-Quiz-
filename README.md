# 🔐 Cyber Security Quiz App

A modern Cyber Security Quiz Web Application built using HTML, CSS, and JavaScript. Test your cyber security knowledge with multiple-choice questions and get your final score instantly.

## 🚀 Features

- 10 Multiple Choice Questions
- Score Calculation
- Timer
- Progress Bar
- Restart Quiz
- Responsive Design
- Modern Dark Theme

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📁 Project Structure

Cyber-Security-Quiz-App/
│── index.html

│── README.md
│── LICENSE

## ▶️ How to Run

1. Download or Clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your browser.

No installation or database is required.
